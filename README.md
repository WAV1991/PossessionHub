# Possession Hub

Track possessed players and access upcoming features in one hub.

## How to use

Copy and paste this code into your executor:

```lua
loadstring(game:HttpGet("https://raw.githubusercontent.com/WAV1991/PossessionHub/refs/heads/main/source"))()
```
## Features
- Player:
  - Speed.
  - Add Minutes.
  - Reset.
  - Noclip.
  - Anti Mute.
  - Gamemode Vote.
  - Instant Collect.
  - Auto Collect.
  - Chat Log.
  - Redeem Codes.
  - Item Esp.
  - Role Esp.
  - Safe Zone.
  - Kill All (Sheriff).
  - Kill Possessor (Sheriff).
- Emotes: 56+ animations available: Levitate, Thriller, RhythmRiot, and many more.
- Sit Emote: 15+ animations available: Standing, Slouched, Shy, and many more.
- Roulette: Allows you to pick any item from the roulette.
- Shop:
  - Auto Buy Spin.
  - Auto Spin.
  - Jade Contemplation.
  - Jade Shatter.
- Troll:
  - Pizza Sound Spam.
  - Glow.
  - Follow Player.
  - Auto Emote Duo.
